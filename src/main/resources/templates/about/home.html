I'm Bannock, a self-taught Java developer and engineer.

This is my personal website that I built using Spring boot. It utilizes Loki for log aggregation, Grafana for observability, and Zipkin for tracing. The front end is done with plain html/css/js. It was designed with the philosophy that the site should be usable without JS, navigable using only the keyboard, and with a look and feel that's fun/amateur as opposed to the simple card layout with an attempted professional look used by most people nowadays.
